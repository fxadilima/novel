## 📑 Catatan

Gunakan icon 📚 (128218) untuk **Heading 1**, gunakan 📑 untuk semua simbol buku.
Simbol buku digunakan dalam konten sebagai **Heading 2**.

### 🔖 Simbol Bookmark

Digunakan untuk heading level 3.

Simbol 🧾 (`&#129534;`) mungkin bisa kita pakai untuk `<details>`.
